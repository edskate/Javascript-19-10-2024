Como o JAVASCRIPT torna as páginas web mais interativas?

O JavaScript permite tornar páginas mais interativas acessando e modificando o conteúdo e a marcação usados em uma página web enquanto ela é visualizada no navegador.

    1. Acessar conteúdo
    2. Modifica conteúdo
    3. Programar regras
    4. Reagir a eventos

O JavaScript engloba muitas regras tradicionais da progrmamação.
Ele pode fazer a página web parecer interetiva respondendo ao que o usuário faz.
